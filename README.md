# cubus-macrokeyboard
**Designed for comfort and efficiency when you're working at your computer.**

The **cubus-macrokeyboard** is a 4-key programmable macro pad created for the **HackClub Blueprint Event**. It's built with the **XIAO Seed RP2040** for maximum flexibility and ease of customization.

---

## Features:
- **4 Programmable Keys**  
  Customize each key for shortcuts, macros, or custom commands.
  
- **Powered by XIAO Seed RP2040**  
  A compact and powerful microcontroller ideal for small, customizable projects.

---

## Getting Started:
1. **Program the Keys**  
   Flash the XIAO RP2040 with your preferred firmware (e.g., CircuitPython, Arduino, or KMK).
   
2. **Customize the Macros**  
   Set up key functions for shortcuts, automation, or any actions that help improve your workflow.

---

## Hardware:
- **XIAO Seed RP2040** microcontroller
- **4 mechanical keys** (MX-style switches)
- **USB-C connection**
